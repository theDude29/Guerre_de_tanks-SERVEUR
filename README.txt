Pour compiler les sources vous devez installer les bibliothèques irrlicht 1.8.4 en version dev (ou version plus récente) et les modules "network", "audio" et "system" de la SFML 2.3 en version dev (ou version plus récente). Vous trouverez facilement les paquets corespondants dans les dépots de votre distribution, n'oubliez pas de prendre les versions de devellopement sinon ça ne marchera pas.

Il faut paramétrer la redirection des ports sur votre boxe pour qu'elle laisse les clients accéder au serveur.

Je sais que les fichiers sont très mal organisés et que le code et illisible mais j'avais 13-14 ans quand j'ai fait ce projet et tant que JE comprenais le code c'était impeccable ;).
